# Weather Forecast App
- WeatherForecast is a responsive weather application built using React, CSS, and JavaScript, allowing users to search for any city and view real-time weather data including temperature, condition type, humidity, wind speed, and a 3-day forecast.
- The design is inspired by Apple Weather with translucent cards and dynamic background images that change based on current weather conditions.

# Features
	•	Search weather details for any city worldwide
	•	Display of current temperature and weather condition
	•	3-day weekly forecast with icons
	•	Humidity and wind speed indicators
	•	Dynamic background images based on weather type (sunny, cloudy, rainy, snowy, foggy)
	•	Responsive and mobile-friendly design (optimized for iPhone dimensions)
	•	Animated “Welcome” screen with typewriter effect on homepage
	•	Clean and translucent card design inspired by Apple Weather

# Tech Stack
	•	React.js (Frontend framework)
	•	CSS3 (Styling and animations)
	•	JavaScript (ES6)
	•	WeatherAPI.com (Weather data provider)
